Memories

Keep track of your best moments everywhere!

View project on GitHub

Installation

Dependencies:

  • Ubuntu SDK
  • U1db - qtdeclarative5-u1db1.0
  • Ubuntu Mobile Icons - ubuntu-mobile-icons

To test run the following commands

qmlscene qml/memories.qml

You can build your own click package to be installed on your devices just by running

click build Memories/

Features

  • Adaptive layout so that the app looks good on every device
  • Tags support to allow to filter memories
  • HTML instead of simple text for memory content
  • Associate photos with your memories
  • Camera support
  • Different view layouts for the home page
  • Localization support
  • Social Networks sharing
  • Protect your memories with a password
  • Sync with UbuntuOne personal cloud database

Contribution

If you are a programmer you can fix any reported and approved bugs in the GitHub's issue tracker. Reporting bugs is another glad form of contribution, just as glad as translations. If you don't want to code and you don't want to translate but you still want to contribute you can donate a few euros to developers through the 'Donate' link you will find in the app about page.

See the Wiki page for some information.
